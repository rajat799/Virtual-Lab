Features

COURSE INTRODUCTION

The aim of the course is to create a cloud-based virtual lab that centralizes resources for cybersecurity investigators, utilizing various cloud forensic tools and methodologies to enhance their skills. The domains covered in this course will provide a structured framework for addressing complex challenges effectively. These include:

  • Data collection and preservation
  • Incident response
  • Network traffic analysis
  • Memory forensics
  • Malware traffic analysis
  • Vulnerability Assessment

The cloud is fundamental to a virtual lab which is providing computing services over the internet that allow users to access resources without needing physical hardware. Instead of relying on personal computers, cloud services enable users to store and manage their data online.

COURSE SYLLABUS :

In this course, we will explore three key topics such as unauthorized access, data breaching and incident response. Unauthorized access refers to instances where individuals gain access to cloud resources or data without permission. Understanding this concept is essential in cloud forensics, as it enables investigators to identify, analyze, and respond to potential security incidents effectively. By grasping the implications of unauthorized access, students will be better equipped to navigate the complexities of cloud security and forensic investigations. Unauthorized access can be done by multiple methods.These includes

  • Credential theft
  • Exploiting vulnerabilities
  • insider threats
  • Brute force attacks

Data breaching is the process of violating the security of the system to gain unauthorized access for sensetive or protected data.It is crucial for understanding cybersecurity risks and protecting sensitive information.The knowledge of data breaches is essential for compliance and incident response. Additionally, it helps in data security for protecting sensetive information and managing potential threats in today's online environment. The methods of data breaching majorly includes

  • Hacking
  • Phishing
  • Malware
  • Insider threats

Incident response is the process of detecting,containing and resolving security incidents in cloud invironments.It will help in identifying threats,limiting damage, restoring system and analyzing root cause to prevent future breaches. The process of incident response involves some essential steps. i.e.

  • Preparation
  • Detection and analysis
  • Containment
  • Recovery
  • Post-incident review
